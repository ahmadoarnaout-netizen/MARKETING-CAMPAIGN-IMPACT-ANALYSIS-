# 📈 Marketing Campaign Impact Analysis

## 📌 Project Overview
This project analyzes the impact of a marketing campaign on website traffic using statistical analysis techniques in R. The study compares visitor traffic before, during, and after the campaign to determine whether the campaign produced a statistically significant effect.

The project includes hypothesis testing, ANOVA analysis, visualization, and interpretation of results.

---

## 🎯 Objectives
- Analyze website visitor traffic data
- Compare campaign periods statistically
- Determine whether the campaign increased traffic
- Visualize visitor trends
- Apply statistical testing methods in R

---

## 🛠️ Technologies Used
- R
- tidyverse
- ggplot2
- dplyr
- readr
- stats

---

## 📂 Dataset
The dataset contains daily website visitor information across multiple campaign periods:
- Before Campaign
- After Campaign
- Follow-up Period

Main variables include:
- Campaign Period
- Number of Visitors

---

## 📊 Statistical Analysis Performed
### Hypothesis Testing
- Welch Two Sample t-test
- Manual t-test calculation

### ANOVA Testing
- Comparison of all campaign periods

### Post-Hoc Analysis
- Tukey Honest Significant Difference (HSD) Test

### Data Visualization
- Boxplots
- Visitor distribution analysis

---

## 📈 Key Findings
- No statistically significant increase in website traffic was found.
- Differences between campaign periods were minor.
- Observed traffic changes may be due to normal fluctuations rather than the campaign itself.
